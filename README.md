# hlsjs-p2p-free
hls集成P2P功能的前端播放器。

# 接入方法

 ```html
<script src="//cdn.jsdelivr.net/npm/hls.js@latest"></script>
```
with
 ```html
<script src="//cdn.jsdelivr.net/gh/midoks/hlsjs-p2p-free@latest/dist/p2p.min.js"></script>
```


# 服务端
